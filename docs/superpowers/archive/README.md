# Archived specs & plans

Design specs and implementation plans for **completed, shipped, and deployed** work,
moved here to keep `docs/superpowers/specs/` and `docs/superpowers/plans/` clear for
in-flight work. Nothing here is active; each document is a historical record of a feature
that is live in production.

- `specs/` — the design spec for each shipped feature (`brainstorming` output)
- `plans/` — the implementation plan for each shipped feature (`writing-plans` output)

Active (in-progress) specs/plans live one level up in `docs/superpowers/specs/` and
`docs/superpowers/plans/`. Full history (including pre-archive paths) is in git.
